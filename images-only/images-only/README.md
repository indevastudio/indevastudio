## Tips

- **Format:** `.webp` is strongly recommended — smallest file size, fastest loading
- **Compress** images to under 500 KB each using [squoosh.app](https://squoosh.app) (free)
- **Dimensions:** Landscape photos → 1200×800 px · Portrait cards → 700×1000 px
- After uploading to GitHub, the site updates in ~30 seconds automatically
